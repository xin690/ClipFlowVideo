## 功能概述

ClipFlow 是一款基于 AI 的视频二创工具，主要功能包括：

| 功能 | 说明 |
|------|------|
| 视频下载 | 支持 YouTube、TikTok、Bilibili 等平台 |
| 语音识别 | 使用 faster-whisper 进行自动语音转文字 |
| AI 改写 | 使用 DeepSeek/OpenAI 改写字幕为中文解说风格 |
| AI 配音 | 使用 Edge TTS 生成中文配音 |
| 字幕烧录 | 将字幕烧录到视频中 |
| 视频合并 | 替换原视频音频为 AI 配音 |

## 快速开始

### 1. 运行环境

```
dist\
├── ClipFlow.exe      # 主程序
└── ffmpeg\           # ffmpeg 依赖
    ├── ffmpeg.exe
    ├── ffplay.exe
    └── ffprobe.exe
```

### 2. 基本使用步骤

1. **启动程序**: 双击 `ClipFlow.exe`
2. **输入视频链接**: 在 URL 输入框中粘贴视频链接
3. **设置输出目录**: 可通过「设置」→「输出设置」修改
4. **配置 API Key** (可选): 通过「设置」→「API配置」添加 DeepSeek/OpenAI API Key
5. **开始处理**: 点击「开始处理」按钮

### 3. 处理流程

```
视频下载 → 音频提取 → 语音识别 → AI 改写 → AI 配音 → 视频合成 → 输出
```

## 详细功能说明

### 视频下载

- **支持平台**: YouTube、TikTok、Bilibili 等
- **下载格式**: 自动选择最佳质量（通常为 webm/mp4）
- **保存位置**: `{输出目录}/{视频ID}/`

### 语音识别

- **引擎**: faster-whisper (默认)
- **模型**: small (可自定义)
- **语言检测**: 自动检测原视频语言

### AI 改写

- **提供商**: DeepSeek (默认) / OpenAI / OpenRouter
- **改写风格**:
  - 口语化、接地气
  - 增强情绪表达
  - 每句 5-15 字
  - 添加语气词增强感染力

**示例**:
```
原字幕: "This is a beautiful sunset"
AI改写: "哇，这夕阳太美了！"
```

### AI 配音

- **语音**: 微软 Edge TTS (中文女声-晓晓)
- **格式**: MP3
- **保存位置**: `{视频目录}/tts_output.mp3`

### 字幕烧录

- **格式支持**: SRT (优先) / ASS
- **烧录方式**: ffmpeg 重新编码
- **字幕样式**: 白色字体，带描边

### 视频合成

- **编码**: H.264 (libx264)
- **音频**: AAC 192kbps
- **输出**: `{视频ID}_merged.mp4`

## 设置配置

### API 配置

| 选项 | 说明 |
|------|------|
| LLM 提供商 | 选择 DeepSeek/OpenAI/OpenRouter |
| API Key | 输入对应的 API Key（可选，不填则使用环境变量） |
| 模型 | 选择使用的模型 |

### 配音设置

| 选项 | 说明 |
|------|------|
| 配音音色 | 选择语音（女声/男声） |
| 语速调整 | 调整语速 (-100% ~ +100%) |
| 音量调整 | 调整音量 (-100% ~ +100%) |

### 语音识别设置

| 选项 | 说明 |
|------|------|
| 识别模型 | tiny / base / small / medium / large |
| 运行设备 | CPU / GPU (CUDA) |

### 输出设置

| 选项 | 说明 |
|------|------|
| 输出目录 | 设置处理结果的保存位置 |
| 视频质量 | 选择输出视频质量 |

## 常见问题

### Q1: 视频下载失败

- 检查网络连接
- 确认视频链接有效且可访问
- 部分平台可能需要 Cookie 认证

### Q2: ffmpeg 未找到

确保 `ffmpeg/` 目录与 `ClipFlow.exe` 在同一目录下：

```
dist\
├── ClipFlow.exe
└── ffmpeg\
    ├── ffmpeg.exe
    └── ...
```

### Q3: AI 改写失败

- 检查 API Key 是否正确配置
- 检查网络连接
- 确认 API 额度充足

### Q4: 字幕烧录失败

- 某些视频格式可能不支持烧录，会自动回退
- 错误信息可在日志文件中查看

### Q5: 内存不足

处理大视频时可能需要较大内存，建议：
- 关闭其他占用内存的程序
- 使用较小分辨率输出

### 目录结构

```
ClipFlow/
├── dist/                    # 打包输出
│   ├── ClipFlow.exe
│   └── ffmpeg/
├── src/                     # 源代码
│   ├── ui/                  # UI 模块
│   ├── core/                # 核心模块
│   ├── utils/               # 工具模块
│   └── ...
├── build/                   # 构建脚本
│   ├── install_deps.bat     # 安装依赖
│   ├── download_ffmpeg.bat  # 下载 ffmpeg
│   └── build.bat            # 打包脚本
├── docs/                    # 文档
```
