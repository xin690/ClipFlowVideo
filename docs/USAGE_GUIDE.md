## 快速开始

### 首次使用

1. 下载ClipFlow压缩包并解压
2. 确保 `ffmpeg` 文件夹与 `ClipFlow.exe` 在同一目录
3. 双击 `ClipFlow.exe` 启动程序
4. 在设置中配置API Key（可选）
5. 输入视频链接，点击"开始处理"

### 目录结构

```
ClipFlow/
├── ClipFlow.exe      # 主程序
└── ffmpeg/          # ffmpeg工具
    ├── ffmpeg.exe
    ├── ffplay.exe
    └── ffprobe.exe
```

## 功能详解

### 1. 视频下载

支持平台：
- YouTube (youtube.com, youtu.be)
- TikTok (tiktok.com)
- Bilibili (bilibili.com)
- Twitter/X (twitter.com, x.com)
- Instagram (instagram.com)

输入格式：
- 完整视频链接
- 视频ID

### 2. 语音识别 (ASR)

使用 WhisperX 进行语音识别：
- 自动识别视频中的语音
- 生成时间轴字幕
- 支持多种语言（主要优化中文）

首次使用会自动下载模型（约140MB-2.9GB）

### 3. AI改写

将原字幕改写为中文解说风格：
- 口语化表达
- 增强情绪表达
- 保持核心信息
- 适合配音长度

需要配置API Key：
- DeepSeek (推荐)
- OpenAI
- OpenRouter

### 4. 配音生成

使用 Edge-TTS 生成中文配音：
- 多种音色可选
- 可调整语速
- 可调整音量

### 5. 视频合成

- 合并原视频与新配音
- 可选添加字幕
- 导出H.264编码视频

## 配置说明

### API配置

1. 点击 `设置 > 首选项`
2. 选择"API配置"标签
3. 选择LLM提供商
4. 输入API Key
5. 点击"测试连接"验证
6. 保存设置

### 配音设置

1. 选择配音音色
   - 晓晓（女声）- 默认
   - 云希（男声）
   - 晓伊（女声）
   - 云扬（男声）

2. 调整语速（-100% ~ +100%）
3. 调整音量（-100% ~ +100%）

### 语音识别设置

1. 选择识别模型：
   - base（默认）- 推荐首次使用
   - small - 更高准确率
   - medium - 更高准确率
   - large - 最高准确率

2. 选择运行设备：
   - CPU（默认）
   - GPU（需要NVIDIA显卡）

### 输出设置

1. 选择输出目录
2. 选择视频质量
3. 勾选字幕选项
4. 勾选AI改写选项

## 使用流程

### 完整流程

```
输入视频链接
    ↓
开始处理
    ↓
┌─────────────────────────────────┐
│ 1. 视频下载 (10%)               │
│ 2. 音频提取 (30%)               │
│ 3. 语音识别 (50%)               │
│ 4. AI改写 (70%)                 │
│ 5. 配音生成 (85%)               │
│ 6. 视频合成 (95%)               │
│ 7. 完成 (100%)                  │
└─────────────────────────────────┘
    ↓
查看输出文件
```

### 注意事项

1. **网络要求**
   - 下载视频需要网络连接
   - AI改写需要API调用（消耗配额）

2. **存储空间**
   - 视频文件可能较大
   - 建议保留至少5GB空间

3. **处理时间**
   - 取决于视频长度
   - 短视频（5分钟内）：约5-15分钟
   - 长视频：可能需要更长时间

4. **API成本**
   - DeepSeek：便宜
   - OpenAI：中等
   - 按实际使用量计费

## 常见问题

### Q1: 程序无法启动？

**原因**：ffmpeg未找到

**解决方法**：
1. 确保 `ffmpeg` 文件夹在 `ClipFlow.exe` 同目录
2. 确保包含 `ffmpeg.exe`, `ffplay.exe`, `ffprobe.exe`

### Q2: 视频下载失败？

**可能原因**：
1. 网络问题
2. 视频链接无效
3. 视频被删除或私密
4. 平台限制

**解决方法**：
1. 检查网络连接
2. 确认链接正确
3. 尝试其他视频

### Q3: 语音识别很慢？

**原因**：
1. 使用CPU模式
2. 模型较大
3. 视频较长

**解决方法**：
1. 使用GPU加速（需要NVIDIA显卡）
2. 选择更小的Whisper模型
3. 耐心等待

### Q4: API调用失败？

**可能原因**：
1. API Key错误
2. API余额不足
3. 网络问题
4. 触发了限制

**解决方法**：
1. 检查API Key是否正确
2. 检查账户余额
3. 稍后重试
4. 换用其他API

### Q5: 生成的配音不自然？

**原因**：
1. 语速设置不当
2. 原文改写质量

**解决方法**：
1. 调整语速设置
2. 在设置中选择不同音色
3. 手动编辑字幕后重新生成

### Q6: 视频不同步？

**原因**：
1. 配音与原视频时长差异
2. 系统自动调整

**说明**：
- 当前版本会自动调整
- 完全同步需要更多处理

### Q7: 如何批量处理？

**当前版本**：
- 暂不支持批量处理
- 需要逐个处理

**建议**：
- 多次运行程序
- 等待当前任务完成后开始下一个

## 输出文件

处理完成后，在输出目录会生成：

```

├── 原视频标题.wav          # 提取的音频
├── 原视频标题_merged.mp4   # 合并后的视频
├── subtitles.srt          # 字幕文件
└── tts_output.mp3         # 配音文件
```
